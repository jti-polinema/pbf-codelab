Menangani Vue Methods

Quasar menggunakan beberapa modifikasi dari Vue Directives. Directives tersebut dapat diterapkan hampir di semua elemen DOM atau komponen.

Contoh Quasar directive:

<div v-ripple>Klik Saya</div>

Catatan: Directive berarti "penunjuk". Vue directives selalu diawali dengan v- dan Quasar directive selalu diawali dengan q-. Hal ini untuk membedakan kepemilikan komponen dalam menggunakannya di templat HTML.

Kita akan memulai menggunakan Quasar Directives, silakan lakukan langkah-langkah praktikum berikut ini.

  1. Kita buat project quasar baru atau dapat melanjutkan dari codelabs 13 sebelumnya. Jika ingin membuat project baru, maka lakukan perintah berikut pada CMD atau terminal.
quasar create belajar-directives-components
  1. Setelah proses pembuatan project baru berhasil, selanjutnya kita definisikan directive Ripple yang akan digunakan pada file quasar.conf.js

Catatan: Semua directives yang akan digunakan, harus didefinisikan terlebih dahulu pada file quasar.conf.js.

  1. Setelah itu masuk ke folder src > pages pilih file Index.vue ganti kode template seperti berikut.
<template>
  <div class="q-pa-md row justify-center">
    <div
      v-ripple
      class="relative-position container flex flex-center text-white"
      :class="classes"
    >
      Klik/tap saya NIM - Nama
    </div>
  </div>
</template>

Perhatian: Ganti NIM dan Nama Anda masing-masing.

  1. Selanjutnya kode script seperti berikut.
<script>
const colors = [
  "primary",
  "amber",
  "secondary",
  "orange",
  "accent",
  "lime",
  "cyan",
  "purple",
  "brown",
  "blue"
];

export default {
  data() {
    return {
      color: colors[0],
      index: 0
    };
  },

  computed: {
    classes() {
      return `bg-${this.color}`;
    }
  },

  mounted() {
    this.timer = setInterval(() => {
      this.index = (this.index + 1) % colors.length;
      this.color = colors[this.index];
    }, 3000);
  },

  beforeDestroy() {
    clearInterval(this.timer);
  }
};
</script>
  1. Untuk kode style ganti seperti berikut.
<style lang="sass" scoped>
.container
  border-radius: 3px
  cursor: pointer
  transition: background 1.5s
  height: 150px
  width: 80%
  max-width: 500px
</style>
  1. Kemudian silakan save dan run perintah quasar dev untuk melihat hasilnya di browser. Seharusnya tampil seperti pada gambar berikut. Jika area berwarna kuning di klik/tap, maka akan ada efek Ripple.

Perhatian: Copy file dan folder satu project Quasar tersebut ke folder laporan src/14_quasar_directives_components/prak1, lalu screenshot hasil di browser dan dibuat laporan. Kemudian push ke repo Anda masing-masing.

Pada praktikum ini kita akan belajar tentang penggunaan Quasar Components. Silakan lakukan langkah-langkah praktikum berikut ini.

  1. Pertama, sebelum kita menggunakan Button dan Icon, perlu kita definisikan komponen yang dibutuhkan pada file quasar.conf.js

Catatan: Setiap kita akan menggunakan komponen, directives, plugin, atau konfigurasi lain yang berkaitan dengan project Quasar kita, maka perlu didefinisikan semua pada file quasar.conf.js.

  1. Selanjutnya kita buat kode HTML di src > pages > Index.vue seperti berikut.
<template>
  <div class="q-pa-md row justify-center">
    <div class="q-pa-md q-gutter-sm">
      <q-btn color="primary" icon="mail" label="On Left" />
      <q-btn color="secondary" icon-right="mail" label="On Right" />
      <q-btn color="red" icon="mail" icon-right="send" label="On Left and Right" />
      <br />
      <q-btn icon="phone" label="Stacked" stack glossy color="purple"/>
    </div>
  </div>
</template>
  1. Save file quasar.conf.js dan Index.vue tersebut, lalu lihat hasilnya di browser. Seharusnya tampil seperti gambar berikut.
