Codelabs: Pemrograman Berbasis Framework

Modul praktikum digital berbasis codelabs untuk mata kuliah Pemrograman Berbasis Framework semester 6 Program Studi D4 Teknik Informatika, Jurusan Teknologi Informasi, Politeknik Negeri Malang

#01 Pengantar Pemrograman Pemrograman Berbasis Framework

Mampu memahami konsep pemrograman modern javascript dan menerapkan teknik pengelolaan repositori proyek berbasis version control (Git).

#02 Pengenalan ReactJS

Mampu memahami konsep framework ReactJS dan membuat project menggunakan ReactJS.

#03 Components, JSX, dan Props

Mampu menerapkan komponen, JSX, dan props di ReactJS.

#04 Event dan State

Mampu menangani event dan state di ReactJS.

#05 React Router

Mampu memahami konsep react router dan penerapannya.

#06 State Management (Redux)

Mampu memahami konsep redux dan penerapannya.

#07 Context

Mampu memahami konsep Context serta penerapannya.

#08 Ujian Tengah Semester (UTS)

Merancang dan membangun tampilan layout aplikasi sesuai dokumen RPP (Project Based Learning).

#09 Setup Database dan Fetching Data (API)

Mampu melakukan setup database dan fetching data.

#10 Authentication

Mampu membuat Authentication pada aplikasi NextJS.

#11 Static and Dynamic Rendering

Mampu menerapkan Static dan Dynamic Rendering pada aplikasi NextJS.

#12 CRUD Firebase

Mampu menerapkan transaksi CRUD ke basis data Firebase.

#13 CRUD di Laravel

Mampu menerapkan transaksi CRUD di Laravel.

#14 Validasi dan Restful API di Laravel dan React

Mampu memahami konsep validasi dan menerapkan restful api dengan perpaduan Laravel dan React.

#15-16 Proyek (Project Based Learning)

Mampu menerapkan materi praktikum secara komprehensif ke proyek sesuai dokumen RPP.

#17 Project Demo/Presentation

Mampu menyelesaikan final project secara berkelompok melalui Git.
